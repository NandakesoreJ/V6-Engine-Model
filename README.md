# V6 Engine Design using Fusion 360

## Overview

This project presents a fully modeled V6 Internal Combustion Engine designed using Autodesk Fusion 360. The objective of this project was to understand engine architecture, mechanical component design, assembly constraints, and CAD modeling techniques while creating a realistic V6 engine assembly suitable for visualization, simulation, and 3D printing applications.

The engine consists of individually designed components that are assembled to form a complete V6 engine system.
